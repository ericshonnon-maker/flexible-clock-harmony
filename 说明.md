# 弹性下班提醒 - 功能说明

## 一、软件简介

弹性下班提醒是一款专为上班族设计的轻量工具，根据上班打卡时间自动计算下班时间。支持弹性上下班区间、午休排除、迟到处理等规则，实时显示距离下班的倒计时。

本版本基于 Python PySide6 Windows 原版移植，运行于 HarmonyOS 设备。

---

## 二、核心功能

### 2.1 上班打卡

点击主界面底部的**「上班打卡」**按钮，系统记录当前时间作为上班时间，并自动计算下班时间。

打卡后，按钮消失，显示实时倒计时和下班时间预测。

### 2.2 三种下班场景

| 场景 | 判断条件 | 下班时间计算 | 说明 |
|------|----------|-------------|------|
| **早到** | 打卡时间 < 弹性上班开始时间 | 固定为弹性下班开始时间 | 例如 7:30 打卡 → 17:00 下班 |
| **正常** | 弹性上班开始时间 ≤ 打卡时间 ≤ 弹性上班结束时间 | 打卡时间 + 工作时长（+ 午休） | 例如 9:00 打卡 → 18:00 下班 |
| **迟到** | 打卡时间 > 弹性上班结束时间 | 固定为弹性下班开始时间 | 迟到按正常下班处理 |

### 2.3 午休排除规则

开启午休排除后，若上班时间在午休开始之前，且计算出的下班时间跨过午休开始时间，则自动加上午休时长。

**示例**：
- 午休：12:00 - 13:00（1小时）
- 9:00 打卡 → 9:00 + 8小时 = 17:00，17:00 > 12:00 且 9:00 < 12:00 → 跨午休 → 实际下班 18:00

### 2.4 实时倒计时

打卡后，主界面显示距离下班时间的倒计时，精确到秒（HH:MM:SS 格式）。每秒自动刷新。

### 2.5 下班到达提醒

当倒计时归零时，界面切换为**「已到下班时间！」**提示，并伴随绿色闪烁效果，提示用户可以下班。

### 2.6 迟到标签

若打卡时间超过弹性上班结束时间，界面左上角显示红色**「迟到」**标签，提示当天属于迟到场景。

## 三、界面说明

### 3.1 主页面（倒计时卡片）

```
┌─────────────────────────────────┐
│ 弹性下班提醒              ⚙  ✕ │  ← 标题栏
├─────────────────────────────────┤
│                                 │
│      [迟到]   02:35:17         │  ← 倒计时（42号白色字体）
│                                 │
│   上班: 09:15  预计下班: 18:15  │  ← 信息区（打卡后显示）
│                                 │
│      「 已到下班时间! 」        │  ← 到达下班时间后闪烁显示
│                                 │
│         [ 上班打卡 ]            │  ← 打卡按钮（打卡前显示）
└─────────────────────────────────┘
```

- **卡片尺寸**：280 × 180 像素
- **风格**：深色半透明毛玻璃效果，圆角 12 像素
- **标题栏**：支持拖拽移动；⚙ 为设置按钮，✕ 为关闭按钮
- **倒计时**：打卡后每秒更新，显示剩余时间
- **迟到标签**：仅在迟到场景下显示，红色背景

### 3.2 设置页面

进入方式：点击主页面右上角 ⚙ 按钮

可配置项目：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 弹性上班开始时间 | 时间选择 | 08:00 | 早到场景的判断边界 |
| 弹性上班结束时间 | 时间选择 | 09:30 | 正常/迟到场景的分界线 |
| 弹性下班开始时间 | 时间选择 | 17:00 | 早到和迟到场景的固定下班时间 |
| 弹性下班结束时间 | 时间选择 | 18:30 | 正常场景弹性下班的最晚时间 |
| 工作时长 | 数值选择 | 8.0 小时 | 每日应工作时长 |
| 排除午休 | 开关 | 开启 | 开启后跨午休自动顺延 |
| 午休开始时间 | 时间选择 | 12:00 | 午休起始时间 |
| 午休结束时间 | 时间选择 | 13:00 | 午休结束时间 |

---

## 四、使用流程

1. **首次使用**：打开应用 → 点击「上班打卡」→ 开始计时
2. **日常使用**：上班到岗后打卡 → 随时查看倒计时 → 下班时间到后留意闪烁提醒
3. **修改设置**：点击 ⚙ → 修改参数 → 保存
4. **次日使用**：应用自动在午夜清除前一日记录，次日打卡不覆盖历史

---

## 五、技术特性

- **跨平台**：基于 HarmonyOS ArkTS，支持手机、平板设备
- **轻量卡片**：280×180 小窗口，不遮挡工作视线
- **数据持久化**：配置和打卡记录存储在本地，重启不丢失
- **自动跨天**：每日凌晨自动重置打卡记录
- **深色主题**：适配深色模式，护眼低干扰

---

## 六、注意事项

1. 打卡记录以日期为界，跨天后自动清除，无需手动重置
2. 迟到场景按弹性下班开始时间处理，不顺延工作时长
3. 午休排除仅在「上班时间 < 午休开始」且「计算下班时间 > 午休开始」时触发
4. 应用窗口可通过拖拽标题栏移动位置
5. 关闭按钮（✕）仅隐藏窗口，不退出应用后台运行

---

## 七、默认参数参考

```
弹性上班区间：08:00 - 09:30
弹性下班区间：17:00 - 18:30
工作时长：8 小时
午休：12:00 - 13:00（排除）
```
